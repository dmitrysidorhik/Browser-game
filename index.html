Knocked down bricks

The development of this game is intended for teaching web development
Game description: When you first start the game, the message "NEW GAME" appears. The player for the entire period of the game is set to 5 lives. After knocking down the last brick, there is a transition to the next level with the appearance of the corresponding inscription and the output of the current level. When moving to the next level, the speed of the ball increases, and there is also an increase in the speed of the beater. When all lives are lost, the inscription "GAME OVER" is sent. The player is given a choice: Restart or stop. When you select restart, the initial values ​​​​of the level, speed and lives are established. When you click on the "stop" button, the game stops, as well as the output of information about the total number of knocked down bricks and the total elapsed time.
